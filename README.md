# Threads Clone using MERN Stack

Based on the YouTube video by AsAProgrammer https://www.youtube.com/watch?v=G4V4xO9wyD8

## Frontend

-   React
-   Chakra UI
-   TailwindCSS

### Run

```bash
npm run dev
```

## Backend

-   Node.js
    -   Express
    -   jsonwebtoken
    -   bcrypt
    -   dotenv
    -   mongoose
    -   cookie-parser
    -   nodemon (dev)
-   MongoDB
